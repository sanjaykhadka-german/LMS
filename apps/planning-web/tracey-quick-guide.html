Tracey — Quick Start Guide

German Butchery Production Planning · High-level overview

Login & Navigation

Getting into Tracey and finding your way around the app.

1

Open the app

Go to tracey-planning-app.vercel.app in any modern browser. Bookmark it for quick access.

2

Sign in

Enter your email address and password. If it's your first time, use the link in your invitation email to set your password.

Forgot your password? Use Forgot password on the login screen to reset via email.
3

The sidebar

Everything lives in the left sidebar. The main sections are:

  • Dashboard — at-a-glance overview
  • Item Master — all products and ingredients
  • BOMs — recipes and formulations
  • Suppliers — supplier list and spec docs
  • Settings — departments, categories, allergens, users
Click the collapse button (panel icon, top of sidebar) to fold the sidebar into a narrow icon rail — useful on smaller screens.
4

User roles

Tracey has two roles. Your role controls what you can change:

  • Admin  Full access — create, edit, delete everything including settings and users.
  • Staff  View and edit day-to-day content. Cannot change settings or manage users.

Item Master

The central register of every product, ingredient, packaging and WIP in the business.

Item types

Raw Material RM

Purchased ingredients — meat, spices, casings, additives.

WIP / Mix WIP

Semi-finished blends that are used as components in other recipes.

Finished Good FG

End products sold to customers.

Packaging PKG

Bags, trays, labels and other packaging materials.

Key tasks
1

Finding items

Use the filter bar at the top of the Item Master to search by Item Number, Name, Description, Type, Category, Department, or Supplier. Filters combine — type a name and select a category to narrow results. Clear a filter to reset.

2

Adding a new item

Click + New Item (top right). Fill in the required fields: Code, Name, Type and Unit. All other fields are optional and can be filled in later.

Item codes must be unique. Use a consistent format like RM-001 for raw materials or FG-042 for finished goods.
3

Editing items inline (Bulk Edit Grid)

To update many items at once, click the grid/edit icon next to the item count. The table switches to edit mode — click any cell to change it directly. Dropdowns in the grid have a search box: start typing to filter options. Press Esc to cancel, Enter to confirm, or click Save Changes when done.

4

Export & Import

Click Export / Import (top right) to:

  • Export — download the current filtered view as a CSV spreadsheet.
  • Download Template — get a blank CSV with all the right columns.
  • Import — upload a filled-in CSV to create or update items in bulk.

BOMs & Recipes

Bill of Materials — the ingredient formulations that drive production planning.

1

What is a BOM?

A BOM (Bill of Materials) links a finished product or WIP to its list of ingredients with exact quantities per batch. Each product can have multiple BOM versions — useful for seasonal recipes or reformulations.

2

Creating a new BOM

Go to BOMs → + New BOM. Fill in the header:

  • Item — the product this recipe is for (type to search)
  • Reference Batch Size — e.g. 900 kg. All ingredient quantities are relative to this.
  • Yield Factor — accounts for cooking or processing loss. 85% means you need to put in more than you get out.
  • Active Version — tick this to mark it as the current recipe used in production.
3

Adding ingredients

Click + Add Ingredient. In the Component column, type any part of the ingredient code or name — a search dropdown appears. Select the ingredient and the cursor jumps to the Qty / Batch field automatically.

Press Tab to move through Qty → Unit → % → Grind Size → Comment. On the last column of the last row, Tab automatically adds the next ingredient row.
4

Quick-create a missing ingredient

If an ingredient doesn't exist yet, type its name in the Component field and select + New item… from the dropdown. A quick-create panel lets you add the code, name and type on the spot without leaving the BOM form.

5

CSV import & export

Use the buttons at the top of the ingredients section:

  • Template — download a blank CSV with the correct columns to fill in offline.
  • Import CSV — upload a filled CSV to add ingredients in bulk. Matches by item code first, then by name.
  • Export CSV — save the current ingredient list as a spreadsheet for review or copying to another BOM.
6

Approving a BOM

Once a recipe is finalised, click ✓ Approve BOM. Approved BOMs are clearly marked in the list. Click again to unapprove if a recipe needs revision.

Suppliers & Ordering

Manage your supplier register, contacts, spec documents and order splits.

1

Supplier register

Go to Suppliers in the sidebar to see all active suppliers. Each supplier card shows their code, name, contact details and status. Click a supplier to open their full profile.

2

Adding a supplier

Click + New Supplier and fill in the details — code, name, contact name, phone, email and payment terms. Set Active so they appear in dropdowns across the app.

3

Spec documents

Each supplier has a Spec Docs section on their profile page. Upload product specification sheets, safety data sheets or compliance certificates. Documents are tagged as Current or Archived so you always know which version is active.

Spec docs are also visible from the individual item page for any item linked to that supplier.
4

Linking items to suppliers

On each item's detail page you can link one or more suppliers. This lets you filter the Item Master by supplier and drives the order split feature.

5

Order splits

Upload a master order/inventory spreadsheet and Tracey splits it into one sheet per supplier — filtered to only the items that supplier provides. Each split sheet includes an Ordered checkbox and a link back to the full register, making it easy to tick off orders as they're placed.

You're all set! For further help contact your system administrator.
